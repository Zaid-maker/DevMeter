# DevMeter

DevMeter is a coding time tracker that automatically records your coding activity and syncs it with your dashboard.

## Features

- **Automated Tracking**: Automatically track your coding time as you work.
- **Language Breakdown**: See which languages you spend the most time on.
- **Project Tracking**: Organizes your stats by project.
- **Dashboard Integration**: Syncs seamlessly with the DevMeter web application.

## Getting Started

1. Install the DevMeter extension in VS Code.
2. Open the command palette (`Ctrl+Shift+P`).
3. Run `DevMeter: Enter API Key`.
4. Paste your API key from the DevMeter dashboard.

## Configuration

You can configure the following settings in VS Code:

- `devmeter.apiKey`: Your personal API key.
- `devmeter.apiUrl`: The URL of your DevMeter backend (default: `https://dev-meter.vercel.app/api`).
